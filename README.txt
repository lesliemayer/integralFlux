Instructions for running SEISS integral flux algorithm
test cases:
======================================================

To compile & run all the test cases :

> ./run_test_cases.bash

To plot results of performance test cases :

> ./run_plot_test_cases.bash

Directories:

Where C++ code and makefile is :

./SEISS_INTEGRAL_FLUX/C++/     

Where idl plotting code is :

./SEISS_INTEGRAL_FLUX/IDL/

Where all data is :

./SEISS_INTEGRAL_FLUX/DATA/

Where test case files are :

./SEISS_INTEGRAL_FLUX/DATA/DIAGNOSTICS/INPUT/

Where output of test run goes :

./SEISS_INTEGRAL_FLUX/DATA/DIAGNOSTICS/OUTPUT/

Where comparison plots of performance tests go :

./SEISS_INTEGRAL_FLUX/DATA/DIAGNOSTICS/PLOTS/

Where GOES8, GOES10, GOES11 integral flux measurements are 
(used for performance comparison plots) :

~/SEISS_INTEGRAL_FLUX/integral_flux/V6.13/DATA/ORIGINALGOES/

Where baseline GOESR integral flux files are (use these files
to compare with the test run output files) :

~/SEISS_INTEGRAL_FLUX/integral_flux/V6.13/DATA/BASELINE/OUTPUT/
